## 🧾 Dependencies

- [TTY::Reader](https://github.com/piotrmurach/tty-reader) to get keypress.

## ⌨️ Usage

Download the file below then load it in `irb`:

```rb
require_relative "path/to/game2048"
```

Then start the game:

```rb
Game2048.new.play
```
